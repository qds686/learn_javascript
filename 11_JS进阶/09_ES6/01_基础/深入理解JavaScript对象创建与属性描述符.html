Object.create() 与 Object.getOwnPropertyDescriptors()

深入理解JavaScript对象创建与属性描述符

Object.getOwnPropertyDescriptors()

该方法返回指定对象所有自身属性的描述符（包括可枚举和不可枚举属性）。

属性描述符包含以下信息：

  • value: 属性的值
  • writable: 是否可写
  • enumerable: 是否可枚举
  • configurable: 是否可配置
  • get: getter函数（如果存在）
  • set: setter函数（如果存在）

示例代码：

var obj = { a: 1, b: 2, c: 3 };
console.log(Object.getOwnPropertyDescriptors(obj));

输出结果：

Object.create()

该方法创建一个新对象，使用现有的对象来提供新创建的对象的__proto__。

语法：Object.create(proto, [propertiesObject])

参数：

  • proto: 新创建对象的原型对象
  • propertiesObject: 可选。属性描述符对象

在示例中，我们使用Object.getOwnPropertyDescriptors()获取原对象的所有属性描述符，然后使用Object.create()创建一个具有相同原型和属性的新对象。

示例代码：

// 使用Object.create()和Object.getOwnPropertyDescriptors()
// 创建一个完全相同的克隆对象
const clone = Object.create(
    Object.getPrototypeOf(obj),
    Object.getOwnPropertyDescriptors(obj)
);
console.log(clone);

输出结果：

代码含义解析

这段代码的含义是：

  1. 首先创建一个对象 obj，它有三个属性：a、b 和 c
  2. 使用 Object.getOwnPropertyDescriptors(obj) 获取 obj 所有自身属性的描述符
  3. 使用 Object.create() 方法创建一个新对象：
    • 第一个参数 Object.getPrototypeOf(obj) 获取 obj 的原型，确保克隆对象有相同的原型链
    • 第二个参数 Object.getOwnPropertyDescriptors(obj) 将原对象的所有属性（包括它们的特性如 enumerable、writable、configurable）复制到新对象

这种方法创建的克隆对象会保留原对象的所有属性特性，是一种"浅拷贝"的高级形式。
